# OWNERSHIP ANALYSIS FRAMEWORK

**Detection Parameters**
- Capture all disclosed ownership interests and control structures
- Record both equity stakes and governance arrangements
- Map ownership structures across organizational hierarchies

**Corporate Structure Categories**
- Operating Companies: Document shareholder distributions
- Fiduciary Entities: Identify principals and beneficiaries
- Market-Traded Organizations: Record listing and regulatory status

**Relationship Mapping Approach**
- Multi-tier ownership: Calculate cumulative interest through layers
- Parallel ownership routes: Aggregate all connection paths
